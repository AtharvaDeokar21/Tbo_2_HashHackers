3️⃣ Core Modules

🔥 A. Trend Intelligence Engine
Purpose:
Calculate a demand score for a destination using multiple signals.
Signals Used:
Booking surge %
Search spike %
Price change %
Volatility index
Momentum indicator
Output:
{
  "destination": "Dubai",
  "booking_surge_percent": 18.4,
  "search_spike_percent": 22.1,
  "avg_price_change_percent": -3.2,
  "volatility_index": 0.34,
  "momentum_indicator": "Rising",
  "final_score": 0.62
}

Logic:
Weighted scoring model combining normalized demand metrics.

🎯 B. Targeting Engine
Purpose:
Identify highest-value audience segment for that destination.
Inputs:
customer_rfm_scores
customer_engagement
past itinerary interactions
Output:
[
  {
    "segment": "High Value",
    "score": 0.87
  },
  {
    "segment": "Family",
    "score": 0.64
  }
]

Logic:
Weighted scoring using:
Recency
Frequency
Monetary value
Engagement rate
Destination affinity

📦 C. Campaign Builder
Purpose:
Generate structured marketing plan.
Output:
{
  "positioning_angle": "Skyline Luxury Experience",
  "content_plan": {
    "headline": "...",
    "subtext": "...",
    "cta": "..."
  }
}

Uses Groq LLM for:
Positioning narrative

Tone personalization

5️⃣ API Endpoints

1️⃣ Get Trend Data
Endpoint:
GET /api/trend/<destination>

Example:
GET /api/trend/Dubai

Response:
{
  "destination": "Dubai",
  "trend_score": 0.62,
  "raw_data": {
    "booking_surge_percent": 18.4,
    "search_spike_percent": 22.1,
    "volatility_index": 0.34,
    "momentum_indicator": "Rising",
    "final_score": 0.62
  }
}


2️⃣ Get Target Segments
Endpoint:
GET /api/targets/<destination>

Example:
GET /api/targets/Dubai

Response:
{
  "destination": "Dubai",
  "top_segments": [
    {
      "segment": "High Value",
      "score": 0.87
    }
  ]
}


3️⃣ Launch Full Campaign
Endpoint:
POST /api/campaign/launch

Body:
{
  "agent_id": "uuid-agent-id",
  "destination": "Dubai"
}

Response:
{
  "destination": "Dubai",
  "trend_score": 0.62,
  "image_url": "https://res.cloudinary.com/...",
  "segment": "High Value"
}


4️⃣ Fetch Campaign History
Endpoint:
GET /api/campaign/<destination>

Response:
{
  "destination": "Dubai",
  "campaigns": [
    {
      "campaign_id": "uuid",
      "trend_score": 0.62,
      "image_url": "https://res.cloudinary.com/...",
      "created_at": "2026-02-21T09:22:00"
    }
  ]
}
